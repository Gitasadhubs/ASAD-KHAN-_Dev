# Asad Ashraf Portfolio

Personal DevOps portfolio website built with plain HTML + CSS.

## Focus
- DevOps Engineer profile and contact details
- Cloud, CI/CD, Kubernetes, and Infrastructure-as-Code skills
- Project highlights from real deployment work

## Files
- `index.html` - page structure and content
- `h.css` - styles and responsive layout
- `asad-ashraf-devops-cv-2026.pdf` - downloadable CV from the hero section
- `abd.png`, `s.jpg`, `s.logo.png` - image assets
- `vercel.json` - Vercel deployment headers and caching rules

## Run locally
Open `index.html` in a browser.

## Deploy on Vercel
1. Push this folder to a GitHub repository.
2. Import the repository in Vercel.
3. Keep framework preset as `Other` (static site).
4. Leave build command empty and output directory empty.
5. Deploy.
